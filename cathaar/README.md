# CatDetection
Detecting cats in images with OpenCV

## Getting Started

### Prerequisites

What things you need to install:

* Python 
* OpenCV 

## To Run 

1. Clone this repository:
```
git clone https://github.com/haribaskar/CatDetection-HaarCascade.git
```

2. Run:
```
python cat_detector.py -i images/cat_04.jpg
```
